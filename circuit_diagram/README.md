# 回路図エディタ

電気回路の部品をUI上で配置し、端子や空白点を使って配線し、完成した回路図をPNGまたはPDFとして出力できる静的Webアプリです。可変抵抗などの基本部品にも対応しています。

## 使い方

1. `index.html` をブラウザで開きます。
2. 左の部品パレットから部品をクリックして追加、またはドラッグしてキャンバスへ配置します。
3. 選択モードで部品を移動します。判定範囲は記号より広めです。
4. 表示中の部品名や値は、文字そのものをドラッグして位置を動かせます。
5. 選択モードでは、Commandクリックや空白からの範囲ドラッグで複数選択できます。選択中の部品は複製できます。
6. 配線モードで端子、導線、または空白をクリックし、上下左右方向の直線として配線します。
7. `回路中に文字を非表示` で、回路上の部品名と値の表示を切り替えます。初期設定では非表示です。
8. 右側の設定で部品名、値、図面タイトル、メモを編集します。新規部品は `R_1` や `C_1` の形式で連番が付き、数字部分は下付きで表示されます。`"R"_1` のように入力するとアルファベットは通常体で表示されます。
9. 右上の `PNG` または `PDF` ボタンで書き出します。

PDF書き出しはグリッドや画面UIを含めず、回路の範囲に合わせたベクターPDFとして出力します。部品端子や導線端の接続点は表示せず、T字分岐の交点だけ黒点で示します。

## 操作

- `R`: 選択中の部品を回転
- `Command + D`: 選択中の部品を複製
- `Delete`: 選択中の部品または配線を削除
- `Esc`: 選択や配線中の操作を解除
- `クリア`: 回路を消し、部品の連番も1番からに戻す

依存ライブラリは不要です。
